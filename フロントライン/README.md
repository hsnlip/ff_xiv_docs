# フロントライン
基本的なところは[公式サイト](https://jp.finalfantasyxiv.com/lodestone/playguide/contentsguide/frontline/)を見て。  

ざっくり言うと合計７２人で行う大規模PvPモードのこと。  
７２人を２４人ずつ、３つのグランドカンパニーに分けて、３すくみで戦う。  

パッチ6.3時点で、４種類の戦場がありそれぞれ勝利条件などの異なるルールで戦うことになるわよ。  
戦場は日ごとにルーティンしていて、以下の順番になっている。  
リンク先は公式サイトの説明。

+ [外縁遺跡群（制圧戦）](https://jp.finalfantasyxiv.com/lodestone/playguide/contentsguide/frontline/2/)
+ [シールロック（争奪戦）](https://jp.finalfantasyxiv.com/lodestone/playguide/contentsguide/frontline/3/)
+ [フィールド・オブ・グローリー（砕氷戦）](https://jp.finalfantasyxiv.com/lodestone/playguide/contentsguide/frontline/4/)
+ [オンサル・ハカイル（終節戦）](https://jp.finalfantasyxiv.com/lodestone/playguide/contentsguide/frontline/5/)

# 報酬
とりあえず動機を持ったりモチベを維持するために報酬を知っておくといいわね。  

## 全般

### PvP シーズン報酬
マイナーパッチでサイクルされる報酬のこと。  
つまり 6.1, 6.2, 6.3, ... みたいな感じで報酬が更新されるということ。  
PvPをプレイして得られるポイントがたまっていくと、取得したポイントの総数に応じて報酬を得られる仕組み。  
別ゲーでいうところのバトルパス、みたいなもの。  
報酬の内容は大きく分けて２種。  
トロフィークリスタルという戦闘系通貨と、報酬の節目にアイテムなどがもらえる。  
アイテム報酬の内容は過去に「ミニオン」「マウント」「おしゃれ装備」「エモート」などが実装されているわね。  
トロフィークリスタルは「ポートレート素材」「狼の首輪」などに交換できるわよ。  
狼の首輪は「装備」もしくは「対人戦績」に交換できるわね。  

### 対人戦績
最大20000まで保持できる戦闘系通貨の一つ。  
PvPのいずれかのモードをプレイすると得られる仕組み。  
この通貨で過去に実装された「装備（ILあり）」「餓狼装備」「消費アイテム」系統を交換できるわよ。  
食事時間を延ばすスクロールアイテムも交換できるわね。おすすめ。  

### デイリー報酬
コンテンツファインダーのコンテンツルーレットからプレイすると１日１回だけ報酬がおいしい。  
「経験値」「新旧トークン」がもらえる。  
経験値はアライアンスレイドぐらいの経験値。  
デイリーボーナスがない状態でも、そこそこ経験値はいるわよ。  

## アチーブメント報酬
アチーブメントリストのPvPカテゴリから、フロントラインを開くと確認できるわよ。  
称号系は省くけれど、マウントが主たる報酬になると思うわね。  

### グランドカンパニー毎
+ 軍馬の角笛（マウント）  
  各グランドカンパニーに所属した状態で、１００回勝利が条件。  
+ 騎乗システム（マウント）  
  赤いアラグの玉。  
  コンテンツ中に配置されるグランドカンパニーのいずれかにおいて２００勝。  
  もしくはいずれかのグランドカンパニーに所属している間に２００勝。  
  適当に取ったからちょっと詳細は分からず。  
+ 護送システム  
  黄色いアラグの玉。日替わりルールで累計１００勝。  
  （単に１００勝すればよい気がする。）  

### 外縁遺跡群（制圧戦）

### フィールド・オブ・グローリー（砕氷戦）
+ 浮遊システム（マウント）

  累計１００勝報酬。  

# 始めるための準備
これはやらんきゃいけんのとやったほうがええかも、というのをまぜこぜリストアップ。  

## PvPアクションのセット
ウルヴズジェイル係船場でHUDにセットできるわよ。  
PvPアクションは、メインメニューの「マイキャラクター」「PvPプロフィール」「PvP ACTIONS」「JOB」の順にウィンドウを開けば確認、ホットバーに設定できるわね。  
設定できるものは全部設定すること。  
「快気」「浄化」「防御」「軍用エリクサー」「スプリント」はジョブを問わず共通で利用できるから、ジョブ間で共通のホットバーとか、同じ配置にすると使いやすくなるわよ。  

## HUDレイアウトの保存、変更
PvPように一つ、HUDレイアウト設定を作っておくといいわよ。  
PvEとみるところが全然違ったりするから、PvP用に分けておくと便利。  
特によく見るのは、以下の５つ。  
+ PTリスト  
  味方の情報は大事。  
+ キャラクターパラメータ  
  自分のHPとMP情報はもっと大事。  
+ ターゲット情報  
  特にHPとバフ・デバフ情報の二つ。  
  バフ・デバフ情報は効果時間を確認できるから、結構大事。  
+ ステータス情報：バフ  
  自分のバフ。防御の時間を見るときとかに。  
  特定のジョブの場合は、トリガーバフなんかも戦う際に有用ですわね。  
+ 目的リスト/コンテンツ情報  
  PvPのマップ内に入ると、ここに各グランドカンパニーの獲得点数やコンテンツの残り時間が表示されるわよ。  
  どこが有利なのかを確認することで、今どこを攻めたらいいのか、みたいな判断がしやすくなるわよ。  
これらの配置はやりながら随時調整するのがよいと思う。  

## ナビマップの常時表示
ナビマップは邪魔にならない程度にできるだけ大きく、開きっぱなしにすること。  
味方とはぐれないために。また、敵の居場所を知っておくことで危ない場所に行かないように。  

# マップ共通ルール
マップごとのルールに加えて、全マップ共通のルールがあるわよ。  
まぁこれも公式サイトに書いてあるのだけれど、知らないと結構困ることもあるかもしれんので。  

## 戦闘中のプレイヤーのナビマップ表示
ナビマップには、自分の所属するパーティの位置が水色で、ほかの味方アライアンスは緑色で表示されるわよ。  
これはアライアンスレイドとかといっしょ。  
これに加えて、敵と戦闘状態になるとナビマップに位置が表示されてしまうわよ。  
この表示は、そのコンテンツに参加している７２人すべてに表示されるから、軽はずみに戦闘すると位置がばれて奇襲がしづらくなってしまうこともあるわね。  

## 順位によるリミットブレイクの蓄積量の補正
目的リスト/コンテンツ情報で確認できる獲得点数の順位によって、リミットブレイクのたまりやすさに補正がかかるようになっているわよ。  
1位は0.75倍のマイナス補正、2位は補正なし、3位は1.25倍のプラス補正が働く。  

## 戦意高揚
>戦意高揚は、ノックアウトやアシストで増加していく「戦意値」が一定値に達することで自身に付与されるバフステータスです。
>「戦意値」は、最大で「100」まで増加し、「20」増加するごとに、「戦意高揚」のレベルが上昇します。
>「戦意高揚」のレベルに応じて、他者への与ダメージとＨＰ回復量が上昇します。
> なお、蓄積した戦意値は自分が戦闘不能になると半減されます。
> *Ref: https://jp.finalfantasyxiv.com/lodestone/playguide/contentsguide/frontline/4/*

要するに戦闘で敵を倒す、もしくは貢献することで強くなれるということ。  
与ダメージと与回復が最大で50%まで増加するわね。  
「快気」ってこれ対象になるのかしらね。ちゃんと回復量見たことあんまないからわからんわよ。
